# AI Chrome Extension - Article Summarizer

A Chrome Extension that summarizes web articles using Google Gemini API.

## Features
- Summarize any article
- Brief / Detailed / Bullet points mode
- Extracts content automatically
- Uses Gemini AI API

## Tech Stack
- HTML, CSS, JavaScript
- Chrome Extensions API (Manifest V3)
- Google Gemini API

## How to run
1. Clone repo
2. Load unpacked extension in Chrome
3. Add API key in options page
4. Open any article and click summarize
